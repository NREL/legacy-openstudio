Run EnergyPlus


Set Up EnergyPlus

  • Download and install EnergyPlus 5.0 from http://apps1.eere.energy.gov/buildings/energyplus/.
  • Configure Plugins->OpenStudio->Preferences to point to the installed location of EnergyPlus.
  • Download weather files at www.energyplus.gov or use Google Earth to browse the EnergyPlus weather file layer.

Configure Simulation Options

  • Run Control
    • The Run Simulation dialog gives a variety of options for configuring the simulation before running EnergyPlus. These options may result in a different input file than the one you are working with. To see the input file as simulated check in the Run Directory indicated by the dialog.
    • "Run design day simulations" enables you to select whether reports from design day simulations are reported in the detailed (ESO) output file of EnergyPlus. Sizing calculations are still performed, but report data are not output during the sizing period if this box is not checked.
    • "Run weather file simulation" will run a simulation based on an EPW weather file.
  • Weather File Simulation
    • "EPW Path" is required when you have checked the "Run weather file simulation" There are links in the dialog box to download weather files from the EnergyPlus website or as a Google Earth Layer.
    • "Annual simulation" - Your simulation can be run for either a part year or an entire year. If you uncheck "Annual simulation”, you can set the start and end dates.
  • Results
    • Report Annual Building Utility Performance Summary - You can choose the format for this report (HTML, CSV, Tab, orFixed), and the units (SI or IP).
    • Request results in SQLite format - Output is written to a SQLite database and an ESO file. You need this to use the ZeroKit ResultsViewer application for visualizing detailed timeseries output
    • If you want your ESO file in IP units look at the EnergyPlus documentation for the "convertESOMTR" application.
    • Report zone temperatures - Requests 'Zone Mean Air Temperature' and 'Zone Mean Radiant Temperature' report variables.
    • Report daylighting results - Requests 'Daylight Illum at Ref Point' and 'Glare Index at Ref Point' report variables.
    • Report user variables and meters - Preserves user report variables.
    • Report DXF file for verification - EnergyPlus will create a DXF that you can import into SketchUp or another CAD application.
    • Report surface temperatures - Requests 'Surface Inside Temperature' and 'Surface Outside Temperature' report variables.
    • Report zone loads - Requests 'Ideal Loads Air Heating Rate', 'Ideal Loads Air Sensible Cooling Rate', and 'Ideal Loads Air Total Cooling Rate' report variables.
  • Actions on Completion
    • Close shell command window
    • Show error file (ERR) Recommended
    • Show Annual Building Utility Performance Summary (ABUPS)
    • Show report variables and meters file (CSV)

Run EnergyPlus

  • Once the simulation options are set, you can store them by pressing "Apply". Alternatively, you can pressing "Run" to start the simulation.
  • Before running EnergyPlus, OpenStudio will clean up the temporary directory used for simulation and copy in the IDF, IDD, and EPW files for simulation.
  • OpenStudio will then run the ExpandObjectsHVAC preprocessor to convert any HVACTemplate objects to detailed input objects for simulation.
  • OpenStudio will then start an EnergyPlus simulation in a new thread and begin waiting for the simulation to finish. On a Mac, you have to quit the terminal application manually before OpenStudio will see that the simulation is complete.
  • After the simulation finishes, OpenStudio will run the ReadVarsESO program to convert ESO results to CSV format if reporting has been requested.
  • OpenStudio will then copy results (but not the modified IDF file) back to the input directory and rename then from the standard EnergyPlus output filenames to new names based on the input filename. Make sure that you have write access to the directory that the input file is in or this copy will fail.
  • Finally, OpenStudio will open the error file, ABUPS report, and/or CSV file if requested. You should always check for EnergyPlus warning and error messages in the error file.

About The Plugin

 

A few words about the plugin before you dive in (like you haven’t already!)...  This version is a beta—we know there are still some issues to fix.  Below are some notes to clarify exactly what the plugin is and what it is not.

 

What It Is

At this stage of development, the plugin is mainly a geometry editor for EnergyPlus.  The plugin allows you to create a building geometry from scratch:  add zones, draw heat transfer surfaces, draw windows and doors, draw shading surfaces, etc. The best part is that everything is drawn using the standard tools provided by SketchUp.  Once you have something you like, you can save what you have drawn as an EnergyPlus input file. 

 

The plugin also allows you to open any EnergyPlus input file, whether it was drawn with the plugin or not, and begin making geometry changes...modify zones, delete surfaces, add new surfaces, etc. 

 

Beyond geometry editing, the plugin can run EnergyPlus input files and display the results from within SketchUp. 

 

What It Is Not

The plugin is not a full-featured interface for EnergyPlus.  It will not help you create non-geometry EnergyPlus objects, such as materials, constructions, schedules, internal heat gains, HVAC equipment and systems, etc.  In order to effectively use EnergyPlus with the plugin you will still have to be something of an expert in building simulation and must be willing to get your hands dirty working with the low-level EnergyPlus syntax of the text input file.  Or, alternatively, you could use the plugin in conjunction with another third-party interface to make the job easier. 

 

The plugin is also not a translator from SketchUp to EnergyPlus.  You cannot use the plugin to convert an old SketchUp model that you have into an EnergyPlus input file...at least, not without a lot of work redrawing things.  To use the plugin as intended, you must have the energy model in mind from the beginning as you create your SketchUp model. 

 

Conclusion

The plugin is what it is. At the very least the plugin makes it a lot easier to enter the geometry data for your EnergyPlus input files—a task that was once both difficult and time consuming.  We hope that future versions will gradually add the functionality that makes it easier to work with the rest of EnergyPlus as well.  Take a look at the upcoming features for a glimpse of things to come in upcoming versions. 

 

Before you get started we recommend that you take a look at the known issues and FAQ.  If you have problems, feedback, or questions about the plugin, see the support page for information on how to contact us. 

 

Thanks for reading, and good luck with all of your EnergyPlus endeavors!  We hope you find the plugin useful.

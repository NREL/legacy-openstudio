Using The Plugin

 

You can find all commands for the plugin on the Plugins/OpenStudio submenu.  Many of the commands also have equivalent toolbar buttons.  If you don't see any toolbars, go to the View/Toolbars menu and make sure the OpenStudio toolbars are checked. 

 

Starting From Scratch

When SketchUp starts up, there is already a bare-bones template of an EnergyPlus input file associated with the model.  At any time if you are not happy with the way your model is going and you want to start over with a clean slate (at least on the EnergyPlus side), you can select the New command  under the Plugins/OpenStudio menu.  This will get you back to a blank new file template. 

 

As mentioned earlier, its a good idea to already have some constructions in your input file before going any further.  The plugin does not yet manage constructions and materials for you and you will want to have a set of constructions that you can assign to your surfaces. 

 

Now that you are ready to start drawing with SketchUp, the first step is to create a new zone.  Use the New Zone command  to activate the tool that allows you to create and position an empty zone. Once you click somewhere with the tool, you have now created a SketchUp group. The plugin uses groups to separate the geometry of individual zones from each other.  Groups also allow the plugin to separate all of the EnergyPlus-related data and geometry from the rest of the SketchUp model.  Each zone group is associated with a zone object in the input file.  When you add a zone group to the SketchUp model you are adding a zone object to the input file.  See the command summary for a full description. 

 

Detached shading surfaces must also be separated from the rest of the SketchUp model.  But because they are not part of any zone, they must be enclosed in a different type of group, a shading a group.  This group is not associated with any object in the input fileit is only used by the plugin to isolate the detached shading surfaces. Use the New Shading Group command  to activate the tool that allows you to create and position an empty shading group.

 

Drawing New Geometry

After creating a zone group, the next step is to draw your new geometry inside the group. Double-click the zone group to open it for editing.  Once you are inside of the zone group, SketchUp gives you many ways to draw the zone surfaces. 

 

One way to draw a zone is to begin by drawing a face to match the footprint of the zone.  A face can be drawn using the Rectangle tool , Polygon tool , or even the Circle tool .  A face can also be drawn by using the Pencil tool  to draw individual edges until they close the shape.  The plugin automatically recognizes a face drawn on the z=0 plane as a floor surface (gray colored). 

 

 

The footprint face can then be extruded up into the rest of the surfaces of the zone using the Push/Pull tool .  The plugin automatically recognizes vertical faces as wall surfaces (sand colored) and the top face as a roof surface (red colored). 

 

 

Another way to draw a zone is to create all faces using the Pencil tool  to connect individual edges.  That means you can start by drawing a wall first instead of the floor.  The plugin automatically recognizes surfaces as different types based on the direction of their outward normal vector. 

 

 

To create a window sub surface (FenestrationSurface:Detailed.), use the rectangle tool to draw a face inside of another face (usually a wall).  The plugin automatically recognizes the new face as a window sub surface (transparent blue colored) as long as it shares no more than one edge with the base surface and is otherwise completely contained by the base surface and lies in the same plane.

 

 

Another way to create the window is to use the Pencil tool  to draw a rectangular face inside of another face. 

 

To create a door sub surface, the same rules apply as for the window, but make sure the bottom edge of the door face is shared by the base surface.  If you do it correctly, the plugin automatically recognizes the new face as a door sub surface (brown colored). 

 

 

To create an attached shading surface (Shading:Zone:Detailed - an overhang, for example), the best approach is to use the Pencil tool  to draw a rectangle that projects perpendicularly from its base surface. But the attached shading surface cannot share any edges with the base surface. The plugin automatically recognizes the new face as an attached shading surface (light red colored). 

 

 

 

Opening An Existing Input File

One of the nice features of the plugin is that, not only can you create an input file from scratch, you can also open and edit any EnergyPlus input file that you might have already.  Use the Open command  to select the input file that you want to open. 

 

After the file is opened, it's a good idea to check for any problems by selecting the Show Errors And Warnings command. Many common geometry errors (e.g., upside floor or roof, inside out window, non-coplanar surfaces, etc.) are fixed automatically for you just by opening the input file.  Some of these fixes are necessary for the model to draw correctly in SketchUp, so you cannot turn off this feature. 

 

To remind yourself which EnergyPlus input file you are working with, you can select the File Info command to see the file name, the file path, and other details.

In addition to the standard tools that we have discussed, you have almost all SketchUp tools at your service for editing and creating geometry.  The standard edit commands Cut, Copy, and Paste are also available as long as you keep it within a zone group or shading group. 

 

Special Rules For Drawing On z ≠ 0

SketchUp assigns an outward normal vector to each face that is drawn.  OpenStudio uses this vector to determine whether each surface is a floor (downward normal) or roof/ceiling (upward normal).  SketchUp assumes that all surfaces on the z = 0 plan have downward normals.  This is convienent as most surfaces drawn on this plane are indeed intended to be floors.  However, SketchUp uses the order that vertices are drawn (using the right hand rule) to determine the outward normal for surfaces drawn on planes other than z = 0.  This matters when trying to use the push/pull tool to extrude a surface to create walls and a roof.  If the extruded surface is not on the z = 0 plane and the vertices are drawn such that the outward normal is up, OpenStudio will assume that this surface is a roof. When the surface is extruded up, the bottom surface will remain a roof which is not likely what you want to have happen.

Editing Input Object Fields

Not all of the input fields for EnergyPlus geometry objects are handled by the SketchUp drawing tools. The SketchUp tools take care of the 3-D coordinates of the vertices, but they do not give access to the other input fields such as the surface type, construction, boundary conditions, etc.  The plugin attempts to intelligently default most of these input fields for you. However, you will still have to override the defaults sometimes.  For this purpose, the plugin has the Object Info command which displays a dialog window that allows you to edit the input fields directly.  See the command summary for a full description. 

 

You will need to use the Object Info window to set a construction for every surface that you draw. You will also need to use it to set boundary conditions for interzone surfaces. Optionally, you can use it to change the name of a surface or zone. 

 

Helpful Tips For Drawing EnergyPlus Geometry

Here are some helpful tips for drawing EnergyPlus geometry with the plugin in the SketchUp environment:

 

  • Occasionally when you draw a face, SketchUp will guess the wrong side to be out.  You can tell because the inside of the face is a lighter color than the outside of the face.  To flip the face right-side-out use the SketchUp command Edit/Face/Reverse Faces, also found on the context menu. 
  • Use the Outliner window (Window/Outliner) to view a list of all of your zone groups and shading groups. Groups can be selected by clicking items in the outline.  Groups can be opened for editing by double-clicking items in the outline.
  • Use the command View/Component Edit/Hide Rest Of Model to hide all other groups (and the rest of the model) when you are editing a group.  When used with the Outliner, this command helps you get to zones that are buried in the core of a building. 
  • When drawing two or more zones with interzone surfaces, it is easier to draw the zones apart from each other, set the interzone connections with the Object Info window, and then move them into final position using the Move tool. 
  • To position vertices with precision or create surfaces with exact dimensions, use the Value Control Box (VCB) in the lower right corner or use construction lines.  Read your SketchUp documentation.
  • To draw quickly and accurately, use SketchUps inferencing capabilities.  For example, to match the x-, y-, or z-position of the endpoint of a line, hover the cursor over the endpoint until the tooltip appears. Now when you start drawing a parallel line, SketchUp will let you know when you are exactly lined up with the other endpoint.  Also to lock an inference along an axis, hold down the shift key while you draw. Read your SketchUp documentation.

 

Running EnergyPlus

Once you are satisfied with your input file, you can run an EnergyPlus simulation from within SketchUp. Use the Run Simulation command to display the dialog window that allows you to select various run control options and launch the simulation.  See the command summary for a full description.  Before you run EnergyPlus, keep in mind that there is usually a lot more that goes into an input file besides just the geometry objectsthings like internal heat gains, schedules, HVAC equipment and systems, etc.  You will either have to work with the low-level syntax of the input file directly using a text editor or use another interface to create those other input objects.  The Run Simulation command is most useful when all of your non-geometry inputs are already set up and you are just using the plugin to compare different geometry options for the envelope, for example, building massing, orientation, glazing fractions, etc.

Frequently Asked Questions

 

 

Q:  I thought the plugin was going to be open source?  Where is the source?  How can I participate in development?

 

A:  The source code is in the SketchUp Plugins directory and is licensed under the open source GNU General Public License.  See the file “License.txt” for more details.  The collaborative project website can be found here: http://openstudio.sourceforge.net

 

 

Q:  I saw something about the plugin being able to generate complete EnergyPlus models for ASHRAE 90.1 code compliance awhile back.  What happened to that?

 

A:  That feature is served by the toolbar button link to the EnergyPlus Example File Generator (EEFG). The EEFG web interface allows you to automatically generate ASHRAE 90.1 models (and beyond-code models too) and have them emailed back to you, along with the simulation results. You still will be able to open the models that you get back from the web interface within SketchUp and use the plugin for further geometry editing.  The link to the web interface can be find at the bottom of the page here:  http://www.eere.energy.gov/buildings/energyplus/interfaces_tools.html

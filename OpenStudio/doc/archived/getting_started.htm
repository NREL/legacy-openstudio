Getting Started

 

Installing

If you are reading this, then chances are good that you have already figured out how to install the plugin. Congratulations!  The plugin lives in the SketchUp plugins directory at “Google Sketchup 7/Plugins/OpenStudio”.  The next time you start SketchUp, the program should automatically recognize and load OpenStudio as an extension. 

 

Please note the following system requirements:

Windows XP, 2000, Vista, Windows 7 or Macintosh OS X Leopard, Snow Leopard

Google SketchUp 7.0, or SketchUp 7.1

EnergyPlus 5.0

 

Disabling Or Uninstalling

If you ever want to temporarily turn off or disable the plugin, you can do that by visiting the Preferences dialog (Window/Preferences) in SketchUp and looking for Extensions. Simply uncheck the box next to “OpenStudio”.

 

If you want to completely uninstall the plugin, there is an uninstaller program in the plugin directory at “Google Sketchup 7/Plugins/OpenStudio”.  Just double-click the executable to uninstall. 

 

Learning SketchUp

Because the plugin is very tightly integrated with SketchUp, it is imperative that you familiarize yourself with the tools and capabilities that SketchUp offers.  The more that you know how to do in SketchUp, the easier it will be to make the plugin work for you.  Learning SketchUp can be deceptively easy:  many operations are very intuitive, but the program also has some depth to it—there are tons of advanced techniques that you will only discover after studying the documentation. 

 

SketchUp provides a lot of resources to help you learn the program.  SketchUp Help (the SketchUp User's Guide), Online Help Center, and Learning Center are all accessible from the Help menu. There is also a lot information available on the SketchUp web site (http://www.sketchup.com) or Google SketchUp web site (http://sketchup.google.com), including video tutorials (http://sketchup.google.com/vtutorials.html). 

 

Learning EnergyPlus

If you plan to really use the plugin, it is also critical that you learn how to use the EnergyPlus engine. The plugin is not yet a full-featured interface for EnergyPlus so it is unavoidable that you will have to work with the low-level syntax of EnergyPlus input objects at some point. 

 

Fortunately, EnergyPlus provides a lot of documentation to help you learn.  All of the documentation is in PDF format and is located in the “Documentation” folder wherever you installed EnergyPlus.  A good place to start is the Getting Started document.  The Input Output Reference and Engineering Reference documents are also indispensable resources. 

 

If you need additional assistance while learning EnergyPlus, there are a number of ways you can get help by email here:  http://www.eere.energy.gov/buildings/energyplus/support.html

 

Setting Up

If you want to run EnergyPlus from within SketchUp using the plugin, you will need to make sure you have already downloaded and installed the EnergyPlus engine.  If you don’t have it yet, you can download it here:  http://www.energyplus.gov  (Note: You can still draw and edit EnergyPlus input files without the engine.)  Next, select the Preferences command under the Plugins/OpenStudio menu. Locate the EnergyPlus executable “EnergyPlus.exe” wherever you installed it by clicking the Browse button associated with the EXE Path field. If that is successful, you are now ready to draw and run EnergyPlus input files. 

 

Before you start drawing from scratch, you will want to make sure the input file you are working with has some constructions defined already.  (Unfortunately, the plugin does not yet manage constructions and materials for you.)  The new file template (“New_File Template.idf”) comes with a small set of constructions, but you may also want to edit the file in a text editor to add your favorite constructions there so that they are always available when you use the New command.  Another approach is to first create some constructions in an input file using another tool, or using a text editor, then open it with the plugin to draw the geometry.  You may also create a new construction object at any time using the New Construction Stub command. 

 

Conventions

Throughout this User Guide we use some conventions to make it easier to identify a few things.  You might have noticed already that the names of SketchUp and plugin menu commands are shown in bold case and purple color.  When it's helpful to be more specific about the path to the command, slashes will be used to separate menu name, submenu names, and menu command name.  For example: Plugins/OpenStudio/User Guide  To distinguish EnergyPlus input objects we show the name in bold case and dark blue color and often use all capitals.  For example:  BuildingSurface:Detailed.

 

We also use some conventions for terminology.  In SketchUp every object you draw on the screen is called an entity.  Entities include groups, faces, and edges, among others.  These terms are consist with the SketchUp documentation so they shouldn’t be a surprise if you are already a veteran user.  On the EnergyPlus side, the input data file (IDF) is made up of objects as well.  An EnergyPlus input object is a block of text with data values separated by commas and everything terminated with a semicolon.  The individual data values are called fields. The input objects used for describing the geometry of the building are called zones and surfaces.  If you are an EnergyPlus user, then these terms should be very familiar to you. For the purposes of the plugin, there is a one-to-one relationship between groups → zones and faces → surfaces.

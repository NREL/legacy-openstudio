User Guide

About The Plugin...

 

 

Each command is described below.

 

 

New

The New command creates a new EnergyPlus input file.  The new file is generated from a template that includes several input objects that are required for all EnergyPlus input files.  You can edit the default objects in the template by editing the text file “New_File_Template.idf” located in the directory “Google SketchUp 7/Plugins/OpenStudio”.  Until the file is saved, it only resides in memory and is vulnerable to loss. 

 

This command also closes any EnergyPlus input file that might be currently open.  If there are unsaved changes, you will be asked if you want to save the open file first. 

 

New EnergyPlus objects can be created using the New Zone or New Shading Group tools.

 

 

Open...

The Open command opens an existing EnergyPlus input file and draws it using SketchUp entities: groups, faces, and edges.  The input file is now associated with the current SketchUp model.  Any changes that you make to the SketchUp entities with the standard tools (Move, Push/Pull, Scale, Eraser, etc.) will immediately update the EnergyPlus objects in the input file.  New EnergyPlus objects can be created using the New Zone or New Shading Group tools or by editing an existing EnergyPlus group (zone or shading group). Any new entities that you add inside of an EnergyPlus group will immediately add new EnergyPlus objects to the input file. 

 

 

Close

The Close command closes any EnergyPlus input file that might be currently open.  If there are unsaved changes, you will be asked if you want to save the open file first.  You will also be asked if you want to erase all of the SketchUp entities that are associated with EnergyPlus objects.  If you answer “yes”, all of the zone groups and shading groups will be erased. If you answer “no”, the groups will remain but will be stripped of all of their EnergyPlus objects—in other words, subsequent changes to those entities will no longer have any effect on the input file.

 

 

Save

The Save command saves the EnergyPlus input file to its current path. If the file has never been saved before, a dialog asks you where you want to save the new file. 

 

 

Save As...

The Save As command saves the EnergyPlus input file to a new path.  A dialog asks you where you want to save the file. 

 

 

Show Errors And Warnings   

The Show Errors And Warnings command displays a dialog window with a log of errors and warnings that were generated when an EnergyPlus input file is opened.  Errors and warnings can include common geometry problems or mistakes in the syntax of the input file.  Many common geometry errors (e.g., upside floor or roof, inside out window, non-coplanar surfaces, etc.) are fixed automatically for you just by opening the input file. 

 

 

 

File Info

The File Info command displays a dialog window with information about the EnergyPlus input file that you are working with, including the file name, the file path, the file size (as currently saved), and a count of some of the different types of EnergyPlus objects. 

 

 

 

Simulation Info

The Simulation Info command displays a dialog window which allows you to set the fields for the GlobalGeometryRules and Site:Location objects in EnergyPlus.

 

 

Object Info   

The Object Info command displays a dialog window with information about the EnergyPlus object that is currently selected.  The content of the Object Info dialog changes based on the type of object selected. Select a zone group to get object information about the zone.  Open a zone group for editing (double-click), and select a face to get object information about that heat transfer surface.  Open a shading group for editing and select a face to get object information about that detached shading surface.  With no selection, you get object information about the Building object.  Changing selections with the Object Info dialog open will dynamically update the contents with information about the new selection. 

 

The contents of the Object Info dialog window are divided into three sections:  object inputs, object summary, and object text.  The fields in the object inputs section allow you to directly change the fields of the corresponding EnergyPlus input object. The list of fields depends on the type of object that is selected.  For example, for a typical BuildingSurface:Detailed object you can easily assign a new name, set the surface type, set the construction, set the boundary conditions, etc.   Don’t worry, changing the name of the object will automatically update all other objects that reference it.  Note that any changes that you make are not committed until you click the Apply button! Coordinates for surface vertices cannot be edited from this dialog.  You must use the standard SketchUp tools to modify the geometry. 

 

The object summary section displays useful information about the object such as number of vertices, gross area, net area, percent glazing, etc.  Like the input fields, the content of the summary depends on the type of object that is selected. 

 

The object text section displays the actual text of the input object as it appears in the EnergyPlus input file.  The text can be selected and copied but it cannot be edited directly.  Use the input fields in the Object Inputs section to make changes.  All changes are made instantaneously and will show up immediately in the object text. Note that all of the original comments and formatting of the input file (the “context”) are preserved when you make changes. 

 

 

 

Info Tool

The Info Tool command activates a cursor-like tool that displays information about the face that it is hovering over.  A tooltip momentarily shows the EnergyPlus object text for that face.  The status bar shows the position of the cursor in world coordinates. 

 

 

Run Simulation...   

The Run Simulation command displays a dialog window that allows you to select run control options, select a weather file and run period, and choose how you want your results to be reported.  Hit the Run button to launch a simulation of the current EnergyPlus input file.  The EnergyPlus engine will begin to run in a shell command window.  You may continue to work in SketchUp while the simulation is running.  If you decide not to run the simulation at this time, you can still hit the Apply button to save your changes to the input file. 

 

 

Options for run control let you choose whether you want to run design day simulations or the weather file simulation or both.  You can also select the start and end dates for the weather file run period here. Changing these options will update the RunPeriod object in the input file.

 

Options for reporting results allow you to choose what data you want reported back from the simulation. The option “Report Annual Building Utility Performance Summary (ABUPS)” reports a file that gives a helpful summary of the building energy usage over the run period along with a breakdown of end uses. The option “Report user variables and meters” simply preserves any Output:Variable and Output:Meter objects already in the input file.  If this option is not checked, all of the Output:Variable and Output:Meter objects are stripped from the input file.  The option “Report zone temperatures” adds Output:Variable objects for zone mean air temperature (MAT) and zone mean radiant temperature (MRT).  The option “Report inside and outside surface temperatures” adds Output:Variable objects for all inside and outside surface temperatures.  The two preceding options might be useful for data visualization studies. 

 

There are also options for various actions that should be performed when the simulation has completed. The option Close shell command window” will automatically close the shell window (MS-DOS console).  If this option is unchecked, you will have to manually close the window before any other actions can automatically be performed (e.g., show the error file). The option “Show error file” opens the eplusout.err file in the preferred text editor. We highly recommended that you always check your ERR file after any EenrgyPlus run!  The option “Show Annual Building Utility Performance Summary (ABUPS)” opens the ABUPS file for viewing.  The option “Show report variables and meters file (CSV)” opens the report variable output in comma-delimited (CSV) format.  If any of the Output:Variable options were selected, the EnergyPlus output file is postprocessed into a CSV file whether or not the option to show it is selected. 

 

 

New Zone

The New Zone command activates a tool that allows you to create a new zone.  Use the cursor  to select a point that will become the new zone origin by clicking somewhere in the view. An empty SketchUp group highlighted with a blue bounding box will appear anchored at one corner by a construction point.

 

 

With the cursor still positioned over the construction point, double-click it to begin editing the zone.  Any new SketchUp entities that you draw inside of the group with the standard tools (Rectangle, Polygon, Pencil, etc.) will immediately add new EnergyPlus heat transfer surfaces (BuildingSurface:Detailed, FenestrationSurface:Detailed, and Shading:Zone:Detailed) to the input file.  Floors, walls, roofs, windows, doors, and attached shading surfaces are all automatically inferred from their tilts and positions relative to other surfaces. 

 

 

Note:  The size of the bounding box is completely arbitrary. Once the zone group is open for editing, you can draw anywhere within the view and the bounding box will stretch to fit the entities.  If you accidentally click outside the zone group and the bounding box disappears, simply click on the construction point at the zone origin again to select the zone group and the bounding box will reappear.  Double-click the construction point, or any entity within the group, to open it for editing again. 

 

Note2:  The construction point at the zone origin is only used to hold an empty group open.  Once you have drawn some entities inside the group you can erase the construction point without any problem.  There is also a hidden construction point in the opposite corner of the bounding box. Select View/Hidden Geometry to show it, and then you can delete that construction point too.  It just makes the bounding box fit more tightly for small zones. 

 

 

New Shading Group   

The New Shading Group command activates a tool that allows you to create a new shading group. Shading groups are not an EnergyPlus concept, but are used by the plugin to group and isolate detached shading surfaces from the rest of the SketchUp model.  This tool works almost identically to the New Zone tool, except that any new SketchUp entities that you draw inside of the group will immediately add new detached shading surfaces (Shading:Building:Detailed and Shading:Site:Detailed) to the input file, instead of heat transfer surfaces. 

 

 

New Construction Stub

The New Construction Stub command creates a new Construction object in the input file.  A dialog prompts you to enter the name of the new construction. Keep in mind that the new object is only a “stub”—it does not have any material layers assigned to it yet, so you will have to edit the input file to add material layers to the Construction and create the associated Material objects before your input file will run properly. 

 

 

Rendering Settings   

The Rendering Settings dialog allows users to select the EnergyPlus Standard Output (ESO) file corresponding to their current idf model. Once the ESO is loaded the user can select from available run periods (such as design days or annual simulations) and then from available report or meter variables. Scan the "*.mdd", "*.mtd", and "*.rdd" files to see which meters and report variables are available for your model. Currently, only surface or zone level meters and report variables are supported. Note that the ESO file can become quite large for models with many report variables, parsing this file may be somewhat slow and cause the UI to lock up. If possible, limit the amount of meter and report variable output requested, look at the results of design days only, or shorten the simulation run period to minimize the size of the ESO. There is also an option in the user preferences that allows the user to cache ESO files once they are read into a format that can be parsed more quickly. Note that this option will generate a "*.eso.cache" file in the same directory as the selected ESO file. The cache file may be deleted at any time and will be refreshed when the file timestamp on the ESO file is newer than the cache file.

 

 

Render by Color   

Once the ESO file has been loaded, run period selected, and data variable selected the user may view the model colored by the data value using the Render by Color command. Surfaces or zones are colored by the variable value at a given instant in time. To change the time at which variables are displayed use the SketchUp Shadow Settings control (Window->Shadows). If the time is outside of the current run period then no values will be displayed. The user may display a color bar to reference the absolute values of the variable or use the data value cursor.

 

 

Animation   

Once the ESO file is loaded into memory the user may watch how variables change with time throughout their model by using the Amimation command. The time period to run may be configured and the user may configure the animation to loop continuously. No direct animation export is provided. However, a screen capture tool may be used to record a video for later use.

 

 

Preferences

The Preferences command displays a dialog window with various user preferences for the plugin that are preserved between SketchUp sessions.  This is where you set the path to the executable for the EnergyPlus engine.

 

 

User Guide

This command opens the User Guide help pages (what you are reading right now) in your default web browser.

 

 

About The Plugin...

This command displays a modal dialog window with version and copyright information about the plugin.

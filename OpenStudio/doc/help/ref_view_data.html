Information Tool

The Info Tool command activates a cursor-like tool that displays information about the face that it is hovering over. A tooltip momentarily shows the EnergyPlus object text for that face. The status bar shows the position of the cursor in world coordinates. Below is a screenshot of the information tool being used on a base surface of a zone. The data tool will work on all surfaces and sub-surfaces, but will not identify data for object like Daylighting Controls or Illuminance Maps. You can get the same information for them by opening the Object Information Dialog box while the object is selected. (see next subject for screenshot of this)

Object Information Window

The Object Info command displays a dialog window with information about the EnergyPlus object that is currently selected. The content of the Object Info dialog changes based on the type of object selected. Select a zone group to get object information about the zone. Open a zone group for editing (double-click), and select a face to get object information about that heat transfer surface. Open a shading group for editing and select a face to get object information about that detached shading surface. With no selection, you get object information about the Building object. Changing selections with the Object Info dialog open will dynamically update the contents with information about the new selection.

The contents of the Object Info dialog window are divided into three sections: object inputs, object summary, and object text. The fields in the object inputs section allow you to directly change the fields of the corresponding EnergyPlus input object. The list of fields depends on the type of object that is selected. For example, for a typical BuildingSurface:Detailed object you can easily assign a new name, set the surface type, set the construction, set the boundary conditions, etc. Don't worry, changing the name of the object will automatically update all other objects that reference it. Changes that you make are not committed until you click on the dialog outside of the text field! Coordinates for surface vertices cannot be edited from this dialog. You must use the standard SketchUp tools to modify the geometry.

The object summary section displays useful information about the object such as number of vertices, gross area, net area, percent glazing, etc. Like the input fields, the content of the summary depends on the type of object that is selected.

The object text section displays the actual text of the input object as it appears in the EnergyPlus input file. The text can be selected and copied but it cannot be edited directly. Use the input fields in the Object Inputs section to make changes. All changes are made instantaneously and will show up immediately in the object text. Note that original comments and formatting of the input file are preserved when you make changes.

Outliner (Native SketchUp Feature)

The Outliner shows you in a hierarchal way, all of the top level and nested components and groups in your model. You can filter what is show, collapse or expand it, and you can also make selections in the model by clicking on these objects. It is useful your OpenStudio workflow for a number of reasons. First, it is a great way to select objects that may be hard to otherwise select, such as a zone with no surfaces in it. It is also a powerful search tool on large projects with many zones. If you make a habit of naming your zones you can make better use of this.

Surface Search (New)

OpenStudio Surface Search tool picks up where the Outliner leaves off. This is a very powerful surface search tool that allows you to search over just about every parameter of your surfaces. The results are selected, and also visually isolated, by hiding all other surfaces. When you are done with your search, you can unhide the rest of the surfaces by clicking "Unhide All" on the Surface Search dialog. Before searching on a large IDF you can speed up the surface matching by closing the Outliner and Object Information windows.

Set Default EnergyPlus Constructions (New)

The Default Constructions dialog box is a powerful tool that lets you set default constructions for different types of EnergyPlus Surfaces. Default Constructions are used to set constructions on new geometry as it is drawn.

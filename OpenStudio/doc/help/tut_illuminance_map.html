OpenStudio Documentation: Tutorials







Output Illuminance Maps


Adding an Output:IlluminanceMap

  • Enter into the zone where you want to place the Output:IlluminanceMap. If you try to add Output:IlluminanceMap outside a zone, you will see this alert.


    • (Credit: David Goldwasser/NREL)

  • Click on the toolbar button to activate the tool. Click on the model where you want to insert the Output:IlluminanceMap. When you place the object it will locate 30 inches above the surface, edge, or vertex you click on.
  • After you click, the scale tool will be active. You will see 8 pins you can use to scale. The edges scale in one direction, the corner in two.


    • (Credit: David Goldwasser/NREL)

  • See below to learn how to relocate and resize the Output:IlluminanceMap.
  • You can not add more than one Output:IlluminanceMap object to a zone.
  • To obtain results from EnergyPlus for your Output:IlluminanceMap you need to ensure that the zone has Daylighting:Controls and Lights objects.

Making Changes

  • You can use the Scale Tool to change the size of the Output:IlluminanceMap (see the following figure). You have the same number of gridlines after scaling the Illuminance Map.


    • (Credit: David Goldwasser/NREL)

  • You can use the Move Tool to move the Output:IlluminanceMap.
  • You can use the Object Info Dialog box to change the position and size of the Output:IlluminanceMap.


    • (Credit: David Goldwasser/NREL)

  • You can use the Object Info dialog to change the number of grid points. The texture on the map will reflect the values you have for the grid points. Data are recorded at points where lines cross. The map above has 50x10 grid points, vs. 10x10 grid points on the previous image. EnergyPlus supports up to 2,500 grid points (e.g. 50 × 50 grid points) per Output:IlluminanceMap.

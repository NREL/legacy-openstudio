Adding Internal Gains and Ideal Loads Air System to your model


Internal Gains

  • Internal gains include people, lights, equipment, and infiltration. This dialog does not allow user to view gains in current zones, only allows user to specify values that will be written to the file.
  • Applying internal gains to file removes existing objects of these types from selected Zones.
  • Does not currently include hot water end uses, ventilation air requirements, or exterior loads such as parking lot lighting.
  • You can get a report of the last Zone Loads Report in Comma Separated Value (CSV) format to verify the operations that have been performed.

Ideal Loads Air System

  • For models which are starting from scratch, 'HVACTemplate:Zone:IdealLoadsAirSystem' gives the quickest path for load calcs.
  • Applying internal gains to file removes existing objects of this type from selected Zones.
  • You can get a report of the last Zone Loads Report in Comma Separated Value (CSV) format to verify the operations that have been performed. You can also run a DIFF on the IDF file using a file comparison tool. We have a link to these under Release Notes other tools.

Saved Preferences

  • The default values for internal gains correspond to a general office building. The default schedules are those present when starting a new model from scratch. However the user can save settings for internal gains corresponding to other space types or variations. These saved settings can be reloaded and applied to model other space types or to generate perturbations on the current model.

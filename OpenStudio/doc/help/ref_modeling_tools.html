Create a new EnergyPlus Zone

The New Zone command activates a tool that allows you to create a new zone. Use the cursor to select a point that will become the new zone origin by clicking somewhere in the view. Note that you cannot currently change the zone origin in OpenStudio once it is placed, you can change it in the IDF file directly but this may alter your geometry. An empty SketchUp group highlighted with a blue bounding box will appear anchored at one corner by a construction point.


(Credit: David Goldwasser / NREL)

With the cursor still positioned over the construction point, double-click it to begin editing the zone. Any new SketchUp entities that you draw inside of the group with the standard tools (Rectangle, Polygon, Pencil, etc.) will immediately add new EnergyPlus heat transfer surfaces (BuildingSurface:Detailed, FenestrationSurface:Detailed, and Shading:Zone:Detailed) to the input file. Floors, walls, roofs, windows, doors, and attached shading surfaces are all automatically inferred from their tilts and positions relative to other surfaces.


(Credit: David Goldwasser / NREL)

Note: The size of the bounding box is completely arbitrary. Once the zone group is open for editing, you can draw anywhere within the view and the bounding box will stretch to fit the entities. If you accidentally click outside the zone group and the bounding box disappears, simply click on the construction point at the zone origin again to select the zone group and the bounding box will reappear. Double-click the construction point, or any entity within the group, to open it for editing again.

Note2: The construction point at the zone origin is only used to hold an empty group open. Once you have drawn some entities inside the group you can erase the construction point without any problem. There is also a hidden construction point in the opposite corner of the bounding box. Select View/Hidden Geometry to show it, and then you can delete that construction point too. It just makes the bounding box fit more tightly for small zones.

Create a new EnergyPlus Shading Group

The New Shading Group command activates a tool that allows you to create a new shading group. Shading groups are not an EnergyPlus object so will not be preserved in the IDF file, they are only used by the plugin to group and isolate detached shading surfaces from the rest of the SketchUp model. This tool works almost identically to the New Zone tool, except that any new SketchUp entities that you draw inside of the group will immediately add new detached shading surfaces (Shading:Building:Detailed and Shading:Site:Detailed) to the input file, instead of heat transfer surfaces.

Create a new EnergyPlus Daylighting Controls Object (New)

This tool creates a new EnergyPlus Daylighting Control. This consists of a pair of sensors, that by default will be placed at 30 inches above the surface you click on. You can move both sensors via SketchUp tools or the Object Info Window. You can also turn off the second sensor if you only want one, and there are many attributes to set that feed information to EnergyPlus. The Glare Calculation Azimuth Angle of View Direction also changes the appearance of the object in OpenStudio. You can only have one Daylighting Controls in each zone. To get data out of the sensors in EnergyPlus you also need to include lights in your project, which you can add via the Zone Loads Tool. Create a new EnergyPlus Illuminance Map Object (New)

This tool creates a new EnergyPlus Illuminance Map. This consists of rectangle with a grid representing the data points. By default it will be placed at 30 inches above the surface you click on. You can move and resize the Illuminance Map with the sketchUp move and Scale Tool and with the Object Information Window. You can only have one Illuminance Map in each zone. To get data reported for the map in EnergyPlus you also need to include both a Daylighting Controls Object, and lights in your project, which you can add via the Zone Loads Tool. Add Zone Loads to an EnergyPlus Zone (New)

The Zone Loads tool is a way to add loads from people lights, equipment, and also address air infiltration and HVAC thermostat setpoints and schedules. you can apply these settings across the entire model, or just to a specific selection of zones, or a single zone. Match Surfaces between EnergyPlus Zones (New)

This tool doesn't draw any new geometry, it addresses data connections that have to be made between co-planer surfaces on adjacent zones. Once you have drawn the surfaces you can use this tool to handle the data connections. Before searching on alarge IDF you can speed up the search by closing the Outliner and Object Information windows.

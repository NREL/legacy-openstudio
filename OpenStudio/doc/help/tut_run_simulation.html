Run EnergyPlus simulation from within OpenStudio


Setup EnergyPlus

  • Download and install EnergyPlus 5.0 from http://apps1.eere.energy.gov/buildings/energyplus/.
  • Configure OpenStudio preferences to point to the installed location of EnergyPlus.
  • Download weather files at www.energyplus.gov or use Google Earth to browse the EnergyPlus weather file layer.

Configure Simulation Options

  • The Run Simulation dialog gives a variety of options for configuring the simulation before running EnergyPlus. These options may result in a different input file than the one you are working with. To see the input file as simulated check in the Run Directory indicated by the dialog.
  • "Run design day simulations" allows the user to select whether reports from design day simulations are reported in the detailed (ESO) output file of EnergyPlus. Sizing calculations are still performed, but report data is not output during the sizing period if this box is not checked.

Run EnergyPlus

  • Once the simulation options are set they may be stored for later by pressing "Apply" or the simulation may be started by pressing "Run".
  • Before running EnergyPlus, OpenStudio will clean up the temporary directory used for simulation and copy in the IDF, IDD, and EPW files for simulation.
  • OpenStudio will then run the ExpandObjectsHVAC preprocessor to convert any HVACTemplate objects to detailed input objects for simulation.
  • OpenStudio will then start an EnergyPlus simulation in a new thread and begin waiting for the simulation to finish. On Mac, the user has to quit the terminal application by hand before OpenStudio will see that the simulation is complete.
  • After the simulation completes, OpenStudio will run the ReadVarsESO program to convert ESO results to CSV format if reporting has been requested.
  • OpenStudio will then copy results back to the input directory (but not the modified IDF file) and rename then from the standard EnergyPlus output filenames to new names based on the input filename. Make sure that you have write access to the directory that the input file is in or else this copy will fail.
  • Finally, OpenStudio will open the error file, ABUPS report, and/or CSV file if requested. It is a good idea to always check for EnergyPlus warning and error messages in the error file.

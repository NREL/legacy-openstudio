Selection, Modeling, and Inference


Selecting Geometry

  • At first glance selecting geometry in SketchUp is pretty easy, and it is. But there are a lot of special selection techniques that can save a lot of time that it is worth learning about.
  • Single click selects the face.
  • Double click not only selects a face, but also the bounding edges.
  • Triple click widens the selection to all connected entities.
  • Contextual (right) click will also allow you to select other entities on the same layer, or with the same material.
  • Holding down the "Shift" key will allow you to add to or remove objects from your selection. If you pick a selection that had some selected, and some un-selected, they will all get reversed.
  • The "Control" key only adds to the selection without anything being removed.
  • And holding "Control" and "Shift" together set the selection tool to on remove entities from the selection without adding anything.
  • You don't have to select just by clicking, you can drag a box. However if you create the window from left to right, vs. right to left you get different results.
  • If you draw your selection from left to right only entities that are fully within the box are selected.
  • If instead you draw your selection box from right to left, any objects that are even partially within the box are included in the selection.

Drawing and Modeling

  • While you can use SketchUp without every typing in lengths for objects, it is often very useful to be able to enter exact numbers in the "Measurements" box. You can use this for both creating and altering entities.
  • A nice trick with the measurement box that works with most tools is that you can set the value, and then if you don't like it you can go back and keep changing it until you are happy with the results. This works as long as you don't do anything else in SketchUp until you are done. This will make more sense with the Multiple Copy/Move example.
  • Push/Pull repeat - If you have used the Push/Pull tool to extrude something by a set amount, you can re-apply this value to other surfaces to be push/pulled by just double clicking on the surface with the Push/Pull tool selected. A useful example of this would be to push all of the walls of your zone in by 12 inches. You may want to do this if you modeled off of the outside of the exterior wall,but now want to model to the inside wall.
  • Guide lines are a nice tool when you need to want to create reference lines to lock new geometry to, but don't want to clutter up your model,or alter faces that the construction lines cross. Guide lines are created with the Tape Measure Tool, by clicking on an edge, and then clicking again where you ant the line to be. Inference Engine

  • SketchUp has an extensive inference engine that tries to help snap to useful locations. This is well covered on their online inference documentation.
  • The Shift key is very impotent while you are using inferences, as it can lock you into to your current inference.

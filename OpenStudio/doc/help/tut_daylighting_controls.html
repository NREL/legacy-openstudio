Creating and modifying EnergyPlus Daylighting Controls


Making a new Daylighting Control Pair

  • Enter into the Zone where you want to place the Daylighting Controls.If you are not in a zone you will see this alert. If you don't have any objects to click on to activate the Zone, you can select the Zone in SketchUp's Outliner Window.


    • (Credit: David Goldwasser / NREL)

  • Click on the toolbar button to activate the tool. Click on the model where you want to insert the controls. When you place the sensor it will locate 30 inches above the surface you click on. If you want it at a different height you can change it via the move tool or the Object Info window.


    • (Credit: David Goldwasser / NREL)

  • See (Making Changes) below to learn how to relocate the individual control units once you have placed them.
  • You can't add more than one pair in a single EnergyPlus Zone.
  • The New Daylighting Tool isn't the only way to add EnergyPlus Daylighting Controls to your model (see making copies below).
  • To get results out of EnergyPlus for your Daylighting Controls you also need to make sure lights are part of your energy model. This could be through OpenStudio with the Zone Load tool, but you could also add lights to the IDF file with another application.

Making changes and modifications

  • You can change the position of the two sensors through the Object Info dialog box.


    • (Credit: David Goldwasser / NREL)

  • The Object Info dialog box is also where you change attributes for the sensors.
  • If you change the Glare Calculation Azimuth Angle of View Direction setting, the arrow on Daylighting Control will reflect this change.
  • You can also change the position of the sensors by entering the daylighting object and using SketchUp's move tool to drag the sensors around.


    • (Credit: David Goldwasser / NREL)

Making copies of Daylighting Controls

  • As an alternative to using the New Daylighting Controls Tool, you can copy your controls from one Zone and paste them into another.
  • Don't copy a second pair into the same Zone or you will run into errors when you run your simulation. EnergyPlus only supports one pair for each Zone.

Using OpenStudio to validate your Daylighting Sensors in existing IDF files

  • If you import an IDF into OpenStudio any Daylight Controls it contains will show up in OpenStudio. You can use OpenStudio to quickly check and see if the Daylight Controls are were you expected them to be. If you need to add or move any, then you can make the changes in OpenStudio and save back to your IDF file.

OpenStudio Documentation: Tutorials







Daylighting Controls


Add a Daylighting:Controls Object

  • Enter into the zone where you want to place the Daylighting:Controls object. If you try to add Daylighting:Controls outside a zone you will see this alert.


    • (Credit: David Goldwasser/NREL)

  • Click on the toolbar button to activate the tool. Click on the model where you want to insert the Daylighting:Controls object. When you place the object, it will locate 30 inches above the surface, edge, or vertex you click on.


    • (Credit: David Goldwasser/NREL)

  • See below to learn how to relocate the Daylighting:Controls once you have placed them.
  • You cannot add more than one Daylighting:Controls object to a single zone.
  • To obtain results from EnergyPlus for your Daylighting:Controls, make sure the zone also has a Lights object. You can add lights through OpenStudio with the Zone Load tool.

Make Changes

  • You can switch between one and two sensors and change the position of each sensor through the Object Info dialog box.


    • (Credit: David Goldwasser/NREL)

  • You can change the position of the sensors by double-clicking the Daylighting:Controls object and using SketchUp's move tool to drag the sensors around.


    • (Credit: David Goldwasser/NREL)

  • The arrow on Daylighting:Controls object reflects the Glare Calculation Azimuth Angle of View Direction setting. Rotating the sensors does not change this setting.

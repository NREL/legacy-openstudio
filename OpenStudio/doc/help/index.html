OpenStudio version 1.0.8 help and documentation


Help and documentation are sorted into three categories (see left navigation).

  • The Reference Guide provides a short description and dialog screenshots for all OpenStudio toolbar commands and menu items.

  • The Tutorials provide more details and focus on workflow rather than a specific tool. Some tutorials deal directly with OpenStudio tools; others with native SketchUp tools that you may find useful in your OpenStudio workflow.

  • The last section is the Release Notes. This contains installation instructions, software requirements, known issues, links to external resources, and other content.

About OpenStudio 1.0.8


What It Is

  • Is mainly a geometry editor for EnergyPlus. It enables you to create a building geometry from scratch: add thermal zones, draw heat transfer surfaces, draw windows and doors, draw shading surfaces, etc.
  • Enables you to open EnergyPlus input files, whether or not they were drawn with the plugin, and begin to make geometry changes: modify zones, delete surfaces, add new surfaces, etc.
  • Enables you to work with the daylighting-related input objects Daylighting:Controls and Output:IlluminanceMap. You can also add people, lights, electric and gas equipment, infiltration rates, ventilation, and ideal loads air systems so you can quickly have a running simulation starting from scratch. You can learn more about these under New Features on the Release Notes Page.

What It Is Not

  • Is not yet a full-featured interface for EnergyPlus. It will help you create some nongeometry EnergyPlus objects, but you will still need to work with the EnergyPlus IDF file outside of SketchUp. To effectively use EnergyPlus with the plugin, you will still have to be willing to learn about building simulation and work with the low-level EnergyPlus syntax of the text input file.
  • Is also not a translator from SketchUp to EnergyPlus. You cannot use the plugin to convert an old SketchUp model into an EnergyPlus input file unless you consider how to divide your building into thermal zones. To use the plugin as intended, you must have the energy model in mind as you create your EnergyPlus model. Some of the tutorials cover workflows for converting existing models to EnergyPlus input.

Conventions

  • In SketchUp, every object you draw on the screen is called an entity. Entities include groups, faces, and edges. These terms are consist with the SketchUp documentation, which a veteran user will recognize.
  • The EnergyPlus Input Data File (IDF) is also composed of objects. An EnergyPlus input object is a block of text with data values separated by commas and terminated by a semicolon. Allowable values for each field of the object are defined by the Input Data Dictionary (IDD). The InputOutputReference.pdf file in the EnergyPlus documentation provides details about the meaning and impact of each field of each object.

Conclusion

  • OpenStudio makes it easy to enter geometry data for your EnergyPlus input files—a task that was once both difficult and time consuming. This release also begins to add support for nongeometry features of EnergyPlus. We expect to expand access to these in future releases.
  • Before you start, we recommend you look at the known issues. If you have problems, feedback, or questions about the plugin, see the troubleshooting on the Release Notes page.
  • Thanks for reading, and good luck with all your EnergyPlus endeavors. We hope you find the OpenStudio plugin useful.

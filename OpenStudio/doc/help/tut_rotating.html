Rotating Axes and Cameras


Create Axis to Match Object

  • Often part of your building is rotated at a different angle from the rest. This creates a number of problems, one of which is using drawing tools such as rectangles, and also inferences that tie back to the x and y (red and green) axes.
  • SketchUp can move the axes to a new location and specify a new rotation.
  • Select the Axes tool, and then click where you want your new origin to be.


    • (Credit: David Goldwasser/NREL)

  • Click a point to set the positive side of the red (x) axis


    • (Credit: David Goldwasser/NREL)

  • Click a point to set the positive side of the green (Y) axis.


    • (Credit: David Goldwasser/NREL)

  • Now when you draw a rectangle, it is aligned with your new axis instead of the original axes.


    • (Credit: David Goldwasser/NREL)

Align View with Axis

  • The plan view shows that your building is still rotated. You can use SketchUp to align the view with the axes. Right click on the axis, and choose "Align View".


    • (Credit: David Goldwasser/NREL)

  • This will take you to a plan view, still with perspective on, which you can disable if you want a true plan.


    • (Credit: David Goldwasser/NREL)

Reset Axis

  • Click again on the axis and this time choose "reset" to go back to the origin axes. If you want to save an axis set for later use, you can create a scene that can save this.


    • (Credit: David Goldwasser/NREL)

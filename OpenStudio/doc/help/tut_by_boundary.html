Render by Outside Boundary Condition


  • The typical rendering mode (shown below) in OpenStudio renders surfaces based on the class of the surface. For example, doors have a color, roofs, etc.
  • Render by Outside Boundary Condition instead renders surfaces based on their boundary condition.
  • If a wall and a floor have the same boundary condition, they would be rendered with the same color.


    • (Credit: David Goldwasser/NREL)

  • Here is the same model as shown above, but this time with the render mode set to "by boundary". The tan is "ground" blue is "outside", and the green represents "surface" boundary conditions. this is where surfaces of one zone are matched with surfaces of another zone.
  • If you look at the exterior of a model, you should see a lot of blue. If you want to see what is going on inside, the section cut tool works great. You can also turn on x-ray mode, but it can be tough to see what you are looking at.


    • (Credit: David Goldwasser/NREL)
  • Below is an example with x-ray mode turned on.


    • (Credit: David Goldwasser/NREL)

  • There are 18 different colors for the outside boundary conditions.
  • Some of these are grouped. Four are outside conditions, and 9 are ground conditions.
  • Look at EnergyPlus's documentation to learn more about the boundary conditions.
